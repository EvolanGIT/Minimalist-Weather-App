# Minimalist-Weather-App

## Description

- An elegant and compact but very complete Weather Service Report.

- My main focus during this project was to focus on making my code as dry as possible and use bootstrap for mostly all the design.

## Features

- Current day report with Humidity Index, Wind Speed and Farenheit Temperatures.
- 5 day weather forecast.
- Recent Searches dropdown and searchbar with persisting data.
